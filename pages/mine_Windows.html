Windows 挖矿教程

此教程适合64位 Windows 10 16215.0版本或者更高版本

第一步：安装 Linux 子系统

说明：因为当前直接采用 Windows 挖矿的程序仍处于实验性阶段，挖矿效率较低，因此建议满足 Win10 版本需求的用户，仍通过 WSL（Windows Subsystem for Linux）的方式安装 Linux 子系统，最终实现在 CKB 上挖矿。

  • 确认当前 Win10 系统版本

    通过如何检测 Windows 10 系统版本，确认当前 64位 Win10 的系统版本号大于等于 16215.0 。

  • 打开 WSL 开关

    右击我的电脑，点击属性，依次进入：控制面板 —— 程序 —— 启动或关闭 Windows 功能 —— 找到“适用于 Linux 的 Windows 子系统” —— 勾选前面的方框 —— 点击确认保存 —— 根据系统提示重启电脑。

  • 安装 Linux 子系统

    打开 Windows Store，搜索 Ubuntu，选择版本，点击安装，本教程选择安装的是 18.04 版本的 Ubuntu 子系统。


第二步：挖矿文件准备

  1. 启动 Ubuntu 子系统

    进入开始菜单，点击 Ubuntu 系统，开启一个 Ubuntu 系统终端。

  2. 下载文件

    依次根据实际情况选择地复制下面的代码，然后在 Ubuntu 终端内粘贴（右击即为粘贴），粘贴完成，按下回车，执行命令。

    海外用户：

    wget https://github.com/nervosnetwork/ckb/releases/download/v0.15.0/ckb_v0.15.0_x86_64-unknown-linux-gnu.tar.gz

    wget https://github.com/nervosnetwork/ckb/releases/download/v0.15.0/ckb_v0.15.0_x86_64-unknown-linux-gnu.tar.gz.asc

    国内镜像：

    111

    222

  3. 解压文件包并创建路径

    依次复制并执行以下代码，每输入一行，点击回车键确认执行该条命令。

    tar -xzvf ckb_v0.15.0_x86_64-unknown-linux-gnu.tar.gz

    cd ckb_v0.15.0_x86_64-unknown-linux-gnu

    sudo ln -snf "$(pwd)/ckb" /usr/local/bin/ckb

    sudo ln -snf "$(pwd)/ckb-cli" /usr/local/bin/ckb-cli

  4. 检测当前版本状态

    5. 输入 ckb --version 检查 ckb 版本，确认当前版本为111

    输入 ckb-cli --version 检查 ckb-cli 版本，确认当前版本为111
